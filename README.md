<!-- GETTING STARTED -->
## Getting Started


### Prerequisites

* PostgreSQL database
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/AnimusXov/StudentManagement
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Configure Spring app configuration file in order to connect with your database
